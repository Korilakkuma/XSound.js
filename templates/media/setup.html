setup Returns : MediaModule | MediaFallbackModule

This method setups HTMLMediaElement, media formats and callback functions.

Arguments
TypeDescription
1st Object This argument is plain object that has the following properties.
media HTMLAudioElement or HTMLVideoElement
This value is the instance of HTMLAudioElement or HTMLVideoElement.
formats Array
This value is the array that contains media formats.
callbacks Object
This value is the associative array that contains the callback functions that are defined by HTMLMediaElement.
