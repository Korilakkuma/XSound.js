setup AudioModule

This method setups callback functions.

Arguments
TypeDescription
1st Object This argument is plain object that has the following properties.
decode function
While "decodeAudioData" is invoking, this callback function is invoked.
ready function
When "decodeAudioData" ended, this callback function is invoked.
start function
When audio starts, this callback function is invoked.
stop function
When audio stopped, this callback function is invoked.
update function
While audio is playing, this callback function is invoked.
ended function
When audio ended, this callback function is invoked.
error function
When "decodeAudioData" failed, this callback function is invoked.
