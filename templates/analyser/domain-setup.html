domain#setup Returns : TimeOverview | Time | FFT

This method setups the drawn Canvas or SVG.

Arguments
TypeDescription
1st string This value is the instance of HTMLCanvasElement or SVGElement.

In the case of using Canvas, 


<!-- HTML -->
<div>
    <canvas width="480" height="120" ></canvas>
</div>

In the case of using SVG, 


<!-- HTML -->
<div id="svg-visualizer-parent">
    <svg width="480" height="120" xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink"></svg>
</div>
