domain#setup Returns : TimeAll | Time | FFT

This method setups the drawn Canvas or SVG.

Arguments
TypeDescription
1st string This value is 'canvas' or 'svg'
2nd string This value is id attribute of HTMLCanvasElement or SVGElement
3rd string This value is id attribute of HTMLElement that wraps SVGElement. If Canvas is used, this argument can be omitted.

In the case of using Canvas, 


<!-- HTML -->
<div>
    <canvas id="canvas-visualizer" width="480" height="120" ></canvas>
</div>

In the case of using SVG, 


<!-- HTML -->
<div id="svg-visualizer-parent">
    <svg id="svg-visualizer" width="480" height="120" xmlns="http://www.w3.org/2000/svg" xmlns:xlink="http://www.w3.org/1999/xlink"></svg>
</div>
