setup MML

This method setups callback functions.

Arguments
TypeDescription
1st Object This argument is plain object that has the following properties.
start function
When the MML starts, this callback function is invoked.
stop function
When the MML stops, this callback function is invoked.
ended function
When the MML ended, this callback function is invoked.
error function
When the MML is incorrect, this callback function is invoked.
